Project Title: Education NEP Analytics Dashboard

Summary:
This Power BI project analyzes education data based on the National Education Policy (NEP) framework. The dashboard provides insights into student performance, subject distribution, and institutional metrics to support data-driven decision-making in the education sector.

Key Features:

Interactive dashboard with filters and slicers
Visualization of student performance across subjects
Analysis of marks, totals, and trends
Category-wise and subject-wise breakdown
Dynamic KPIs for quick insights

Tools & Technologies:

Power BI (Data Visualization)
DAX (Data Analysis Expressions)
Data Cleaning & Transformation (Power Query)

Insights:

Identified trends in student performance across different subjects
Highlighted high and low scoring areas
Provided comparative analysis for better academic planning

Use Case:
This dashboard can be used by educational institutions to monitor student progress, improve curriculum planning, and align with NEP guidelines.
